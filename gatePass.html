Gatepass System.

The goal of this project was to create an application that aided the security in managing and documenting visitors within the campus. The application digitized the entire process of documenting and allowing visitors into the campus allowing the campus facilities to implement and use data analytics for better management.
The application was found to increase the efficiency of the process by 17%.
